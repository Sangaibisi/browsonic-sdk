# Code of Conduct

This project follows the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

The full text of the Code of Conduct is available at the link above. Participation in this project — including contributing code, opening issues, reviewing pull requests, and discussion in any project channel — is governed by it.

## Reporting

Reports of unacceptable behaviour can be sent to the project maintainer at **emrullahyildirim@windowslive.com**. Reports are read by the maintainer only and will be handled discreetly. Where appropriate, anonymisation will be preserved when discussing reports with anyone other than the reporter.

## Enforcement

The Contributor Covenant 2.1 enforcement guidelines apply: warnings, temporary bans, and permanent bans are tools of last resort and are used proportionally to the severity and recurrence of the behaviour.
